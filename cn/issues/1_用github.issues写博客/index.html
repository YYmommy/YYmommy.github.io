用GitHub Issues写博客

2022-03-07

blog 实现了自动部署，又纠结起写作工具。手机上喜欢用熊掌记，但是上传发布还是有些麻烦，于是又想起 issues1。受 yihong 的开源博客启发，想到如下的更新流程：

graph TD
A[用 issue 写作] --> |Generate GitBlog README| B[issue 内容添加至 blog 源码仓库]
B --> |Auto Deploy hugo| C[自动生成静态文件并更新 blog 仓库]

一共用到 2 个 action：

  • Generate GitBlog README2
  • Auto Deploy hugo3

相关代码在我的 blog-data 仓库里都可以找到，欢迎参考。
