一个人的周末

2021-09-19

晶和朋友去江之岛，一个人在家折腾了一下博客。这次尝试了 Hugo。很喜欢谢益辉博客的风格，直接不厚道地 Copy 来用了。

步骤参考官方文档：Quick Start。进行到第三步时安装 hugo-ivy 主题：

cd blog 
git init
git submodule add git@github.com:yihui/hugo-ivy.git themes/hugo-ivy
echo theme = \"hugo-ivy\" >> config.toml
